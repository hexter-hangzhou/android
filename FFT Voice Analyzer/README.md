android app 通过mic实时录音 AudioRecord 然后fft，显示频谱，fft使用org.apache.commons.math3.transform.FastFourierTransformer，使用NoiseSuppressor进行自带的噪音抑制。
我设置了buff，3秒循环录音和显示频谱
